\chapter{Resultados}

Os resultados apresentados neste trabalho foram obtidos a partir de
dados simulados (utilizando simuladores de Monte Carlo para colisões
próton-próton) e dados reais medidos no detector (sinais de raios
cósmicos e também dos primeiros ciclos de operação do LHC). Neste
capítulo, inicialmente, serão descritas as características das
diversas bases de dados utilizadas. A seguir, os resultados obtidos
através da aplicação das diversas técnicas de processamento de
sinais propostas neste trabalho serão apresentados.
